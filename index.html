이 사이트는 github_page 저장소에서 관리되는 정적 페이지입니다. 파일을 수정하고 GitHub에 푸시하면 페이지에 바로 반영할 수 있습니다.

관리 방식

이 폴더의 파일을 기준으로 사이트를 관리합니다. 기본 페이지는 index.html에서 시작합니다.

배포 흐름

로컬에서 수정한 뒤 git add, git commit, git push를 실행하면 GitHub Pages로 배포할 수 있습니다.
